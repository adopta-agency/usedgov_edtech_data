About This Project


This project makes the data associated with education technology available in an open and editable format. By using the "Adopta.Agency" framework - that is, adopting Department of Education datasets - federal ed-tech datasets will be cleaned up, standardized, and made machine- and human-readable.

This site uses the Adopta.Agency open data blueprint. That is, it is an open Github template designed to be forked and used to improve government and other public datasets.

You can find more information about this project on the Adopta.Agency website.
